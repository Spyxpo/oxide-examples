# Variables & Types

Learn about variable declaration, type inference, and basic data types.

## Difficulty
Beginner

## Concepts
- Variables
- Type annotations
- Basic types

## Description
This example demonstrates Oxide's variable system:

- Variable declaration with automatic type inference
- Optional type annotations using `variable: type = value` syntax
- Basic types: strings, numbers, booleans
- Type aliases: `i32` (32-bit integer), `f64` (64-bit float)

## Usage
```bash
oxide run main.ox
```

```

Version: 1.0
Awesome: True
```
