# Classes & Objects

Create custom types with classes, methods, and inheritance.

## Difficulty
Intermediate

## Concepts
- Classes
- Constructors
- Methods
- Object creation

## Description
This example demonstrates object-oriented programming in Oxide:

- Class definition with `class`/`endclass`
- Constructor method `init`
- Instance methods
- The `self` keyword for accessing instance properties
- Object instantiation with `new`
- Method chaining

## Usage
```bash
oxide run main.ox
```
