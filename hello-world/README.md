# Hello World

The classic first program - learn the basics of Oxide syntax.

## Difficulty
Beginner

## Concepts
- Print statements
- Basic syntax

## Description
This is the simplest Oxide program. It demonstrates:

- Using the `print` statement to output text
- Basic string literals with double quotes

## Usage
```bash
oxide run main.ox
```

## Expected Output
```
Hello, World!
```
