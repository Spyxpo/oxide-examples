# Functions

Define reusable functions with parameters and return values.

## Difficulty
Beginner

## Concepts
- Function definition
- Parameters
- Return values
- Type annotations

## Description
This example demonstrates how to define and use functions in Oxide:

- Basic function definition with `func`/`endfunc`
- Functions with parameters
- Return statements
- Optional type annotations for parameters and return types
- Function calls

## Usage
```bash
oxide run main.ox
```

```
Hello, Developer!
Sum: 8
Square: 16
Cube: 27
```
